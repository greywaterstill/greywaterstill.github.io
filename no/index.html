Greywater Still

This is a simple project showcasing a concept of a greywater still. The still utilizes the principles of solar distillation to purify greywater for reuse.

Design Concept

The design integrates multiple features for maximizing efficiency in greywater filtration.

Materials Used

  • Solar Panels
  • Glass Cover
  • Reservoir for Water

Steps for Building

The greywater still can be built with the following steps:

  1. Construct the frame.
  2. Install the solar panels.
  3. Setup the water collection reservoir.
